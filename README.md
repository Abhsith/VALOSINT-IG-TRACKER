🚀 Overview

VALOSINT adalah tool Python untuk menganalisa akun Instagram menggunakan data export resmi.

Fitur utama:

- Cek siapa yang unfollow kamu
- Cek siapa yang tidak follow balik
- Monitor pertumbuhan followers
- Kirim hasil ke Telegram

Aman tanpa login dan tanpa scraping.

---

✨ Features

- Detect unfollowers
- Detect not-following-back
- Telegram auto report
- Snapshot comparison
- Profile info (manual cache)
- Export hasil ke file

---

📁 Required Files

Ambil dari Instagram export:

followers_1.json
following.json

---

⚙️ Installation

git clone https://github.com/USERNAME/VALOSINT-IG-TRACKER.git
cd VALOSINT-IG-TRACKER
python tracker.py

▶️ Usage

Jalankan:

1. Create snapshot
2. Compare snapshot
3. Compare + Telegram
4. Add profile cache
